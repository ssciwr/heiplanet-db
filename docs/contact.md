# Contact

## Project Support and Contributions

We welcome questions, feedback, and contributions to the Heiplanet platform. Please reach out through the appropriate channels below based on your needs.

### Technical Support and Software Issues

For technical questions, bug reports, or software-related issues:

- **Email:** [ssc@iwr.uni-heidelberg.de](mailto:ssc@iwr.uni-heidelberg.de)
- **GitHub Issues:** [https://github.com/ssciwr/heiplanet-db/issues](https://github.com/ssciwr/heiplanet-db/issues)

The Scientific Software Center team can assist with:
- Deployment and configuration issues
- Software bugs and errors
- Feature requests
- Development contributions
- API and integration questions

### Research and Scientific Inquiries

For questions about the scientific aspects, disease models, or research collaboration:

- **Hei-Planet Website:** [https://hei-planet.com/](https://hei-planet.com/)

Topics include:
- Disease transmission models
- Data interpretation and analysis
- Research collaboration opportunities
- Scientific methodology questions

### Contributing to the Project

We welcome contributions! To get involved:

1. **Code Contributions:**
   - Fork the repository on GitHub
   - Create a feature branch
   - Submit a pull request with your changes
   - Follow the contribution guidelines in the repository

2. **Data Contributions:**
   - Contact us to discuss new data sources
   - Propose additional disease models or regions

3. **Documentation:**
   - Improve existing documentation
   - Add examples and tutorials
   - Translate documentation

### Reporting Security Issues

If you discover a security vulnerability, please **do not** open a public GitHub issue. Instead, email us directly at [ssc@iwr.uni-heidelberg.de](mailto:ssc@iwr.uni-heidelberg.de) with details.
